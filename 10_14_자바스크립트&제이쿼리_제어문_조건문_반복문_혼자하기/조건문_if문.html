제어문

- 조건문 : 3항 연산자 조건문, if() 조건문, switch() 조건문 (3가지) - 반복문

1) 3항 연산자 조건문

(1) 문법형식 - 조건 ? 참 : 거짓; (2) 합격 불합격 판정 조건문 예시]

2) if 조건문

if ~ 단순 조건문(1가지) 예시 : 성적처리]

if ~ else 단순 조건문(2가지) 예시 : 성적처리]

if ~ else if ~ else 다중 조건문(3가지) 예시 : 성적처리]

if ~ else if ~ else if ~ else if ~ else 다중 조건문 (n가지) 예시 : 학점처리 (A+(95점 이상), A(90점 이상), B+(85점 이상), B(80점 이상), C+(75점 이상), C(70점 이상) D(65점 이상), E(60점 이상), F(60점 미만))]

if 다중 조건문 논리연산 포함(NOT(!), AND(&&), OR(||)) (복잡한 조건 방식)

문제 예시 : 국가검정 정보처리기사 합격 불합격 조건

합격조건 : 5개의 모든 과목이 40점 이상이고, 평균 점수가 60점 이상

3) switch() 조건문

문제 예시 : 메인 버튼의 인덱스 번호에 따라서 URL 바뀜

버튼 번호를 prompt로 입력받아 사이트 주소로 이동하는 조건문

버튼 번호를 click해서 사이트 주소로 이동하는 조건문
